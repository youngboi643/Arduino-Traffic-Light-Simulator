# Arduino Traffic Light Simulator

A simple traffic light simulator built for fun using an Arduino Uno.

## What it does
Cycles through red, red+yellow, green, and yellow with realistic timing, just like a real traffic light.

## Hardware used
- Arduino Uno
- 3 LEDs (red, yellow, green)
- 3 x 220 ohm resistors

## How to run it
1. Wire red LED to pin 2, yellow to pin 3, green to pin 4 (each through a 220 ohm resistor to GND).
2. Open the .ino file in Arduino IDE.
3. Select Tools > Board > Arduino Uno and the correct Port.
4. Click Upload.
